Tabalhando com listas

listas coordenadas

São listas aonde a ordem fazem diferença, podemos usar a tag ol para listas ordenadas e dentro dela podemos usar a tag li para desenvolver elas.
Exemplo:

  1. Acordar
  2. escovar os dentes
  3. alongar o corpo
  4. trocar o tonny
  5. dar a ração dos cachorros

Temos as listas não ordenadas

Listas aonde não importa a ordem como as listas de compras, para isso usamos a tag ul.
Exemplo:

  • Pão
  • Leite
  • Açucar
