Citações

Citações são aquelas frases ditas por utras pessoas, e por serem frases de outros pessoas, devem vir entre aspas, 'e para isso usamos a tag q' um exemplo: O compultador é um burro uito rápido

Citações completas

São citações miores, mais completas, geralmento dando uma melhor explicação,
diferente de uma citação simples que é apenas uma frase, e para isso podemos usar a tag blockquote, e adicionar de onde vem aquela citação, para fazermos isso ussamos a tag cite.
visuaulmente esse tag não faz nada no site, mas para as plataformas como google, bing etc... Eles saberão de onde vem sua citação e isso pode evitar processos por direitos autorais.
Exemplo:
