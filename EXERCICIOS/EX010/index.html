Vamos começar a falar sobre links, ou ancoras

Links e ancoras são hiper-texto texto que tem como objetivo te direcionar para um outro lugar na pagina ou até mesmo uma oputra pagina.

Links externos

são links que te direcionam para uma outra pagina.

veja o exemplo abaixo

Repositoria github da aula

No exemplo acima foi usado um link externo,
ele foi criado com a tag a, que pede uma referencia, deve ser ela a url completa do site,
depois o modo como você quer que ela aparece no site.
Para que o site que vai ser aberto pelo seu link, seja abberto em uma pagina em brando deve se usar o target = '_blank', ou o rel = "external"
