Aula de midias-Videos

Vamos aprender a colocar videos uteis e viaveis em nosso site

Compativeis: mp4, ogv, webn

Nesse modelo eu coloquei o modelo mp4, mas sempre que for montar o site se deve colocar todos os formatos,
para que funcione em todos navegadores,
para fazer isso é só colocar dentro de sorces todos os arquivos que queremos tentar carregar.
Podemos colocar uma imagem de capa no video usando o poster e passando o caminho.
